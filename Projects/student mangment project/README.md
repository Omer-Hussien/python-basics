# 🎓 Student Management System

A simple **Student Management System** built with Python that allows users to manage student records through a command-line interface (CLI).

## 📌 Features

- ➕ Add a new student
- 📋 View all students
- 🔍 Search for a student by ID
- ✏️ Update student information
- 🗑️ Delete a student
- 🚪 Exit the application

## 🛠️ Technologies Used

- Python 3.x
- No external libraries required

## ▶️ How to Run

1. Make sure Python 3 is installed on your computer.
2. Clone this repository or download the project files.

```bash
git clone https://github.com/your-username/student-management-system.git
```

3. Navigate to the project directory.

```bash
cd student-management-system
```

4. Run the Python program.

```bash
python "student mangment project.py"
```

## 💡 How It Works

The program stores student information in a Python list during runtime. Each student record contains:

- Student ID
- Name
- Age
- Major

> **Note:** Data is not saved permanently. Once the program is closed, all records are lost.
