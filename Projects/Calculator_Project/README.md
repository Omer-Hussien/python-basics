# Calculator Project

A simple command-line calculator application written in Python. This program allows users to perform basic arithmetic operations through a menu-driven interface.

## Features

* ➕ Addition
* ✖️ Multiplication
* ➖ Subtraction
* ➗ Division
* 🚪 Exit option

The application displays a menu, prompts the user to select an operation, accepts two integer inputs, performs the calculation, and displays the result.

## Requirements

* Python 3.x

## Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/your-username/calculator-project.git
   ```

2. Navigate to the project directory:

   ```bash
   cd calculator-project
   ```

## Usage

Run the program using:

```bash
python Calculator_Project.py
```

You will see a menu similar to:

```text
========WELCOME TO OUR PROGRAME=======
==This is the calculator application==
--------------------------------------
1. Addition
2. Multiplication
3. Subtraction
4. Division
5. Exit
--------------------------------------
```

Select an option and enter two numbers when prompted.

## Example

```text
Enter your choice: 1
First number: 10
Second number: 5
10 + 5 = 15
```
